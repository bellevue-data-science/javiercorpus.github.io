Hi, I'm Javier Corpus

Data Scientist

I'm a data scientist who loves turning raw data into clear, actionable insights. My work focuses on extracting value from complex datasets and delivering results that drive informed decisions. I also enjoy fine-tuning large language models to tailor them for specific applications, ensuring they deliver precise, domain-relevant results.

Technical Skills

Statistical analysis & inference

I rely on statistical analysis to turn raw numbers into trustworthy conclusions. By applying hypothesis tests, confidence intervals, and regression models, I can quantify uncertainty and verify relationships in the data. These techniques guide my decision-making and help me assess whether a model is performing as expected. A solid statistical foundation keeps my insights grounded in evidence.

Programming (Python/R)

Python and R are the core tools I use for data-science projects. With Python I build pipelines, manipulate data using pandas, and develop models through scikit-learn or TensorFlow; with R I conduct statistical tests and create visualisations. Being fluent in both languages lets me pick the most efficient solution for each problem. I also write clean, reproducible scripts that others can follow and extend.

Machine learning algorithms

I design and implement machine-learning models to generate predictions and discover patterns. I am comfortable with supervised methods such as linear regression, decision trees, and neural networks, as well as unsupervised techniques like clustering and dimensionality reduction. I evaluate model performance using cross-validation, tune hyperparameters, and monitor metrics to ensure reliability. This systematic approach lets me move from raw data to actionable forecasts.

Data wrangling & preprocessing

Before any analysis, I clean and reshape data to make it usable. I handle missing values, normalize formats, engineer features, and merge datasets from different sources. Building automated pipelines for these steps reduces errors and speeds up iteration. Consistent preprocessing ensures that downstream models receive high-quality input.

SQL & database querying

I use SQL daily to extract and aggregate data stored in relational databases. Writing efficient SELECT statements, joins, and window functions lets me retrieve exactly the records I need without overloading the system. I also optimise queries with proper indexing and understand transaction basics for reliable data handling. These skills enable me to work directly with large, production-grade datasets.

Data visualization

I translate complex findings into clear visual stories that stakeholders can act on. Using libraries such as Matplotlib, seaborn, ggplot2, and tools like PowerBI, I craft charts, heatmaps, and interactive dashboards. I focus on presenting trends, outliers, and key comparisons without clutter. Effective visualisation bridges the gap between technical analysis and business decisions.
